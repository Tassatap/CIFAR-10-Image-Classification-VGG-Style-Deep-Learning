# CIFAR-10-Image-Classification-VGG-Style-Deep-Learning
CIFAR-10 image classification using VGG-style CNN with TensorFlow/Keras
CIFAR-10 is a widely used benchmark dataset consisting of 60,000 color images across 10 classes, such as airplanes, cars, birds, and animals. Despite its relatively small image size (32×32), the dataset presents a non-trivial challenge due to inter-class similarity and limited resolution.

To address this, the project adopts a VGG-style architecture, which is known for its simplicity and effectiveness in deep learning. The key idea behind VGG is the use of multiple stacked convolutional layers with small (3×3) filters, allowing the model to learn hierarchical feature representations — from low-level edges and textures to high-level object structures.
